# Plano Operacional Agência Digital: Checklist de Acompanhamento

Esta tabela consolida as principais atividades e sugestões de automação com IA para as fases de Prospecção, Execução do Contrato e Finalização/Renovação, servindo como um guia para o desenvolvimento e implementação.

| Fase | Atividade Principal | Sugestão de Automação/IA | Ferramentas | Status de Desenvolvimento | Status de Implementação | OBS |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **Prospecção** | Identificação de Leads | Pesquisa de empresas com ICP, coleta de dados de contato. | IA (Web Scraping/LLM), Notion | Proposto | Pendente | |
| | Qualificação de Leads | Pré-qualificação de leads, coleta de informações básicas. | Typebot (Agente SDR de IA), n8n | Proposto | Pendente | |
| | Abordagem Direta (Outbound) | Envio de cold emails personalizados, mensagens no LinkedIn. | n8n, LLM (Manus.im/ChatGPT) | Proposto | Pendente | |
| | Follow-ups | Acompanhamento automático de leads. | n8n, Notion | Proposto | Pendente | |
| | Agendamento de Reuniões | Automação do agendamento de reuniões. | Calendly/Acuity Scheduling, n8n | Proposto | Pendente | |
| **Execução do Contrato** | Planejamento de Campanhas | Geração de ideias de segmentação, palavras-chave, copy. | LLM (Manus.im/ChatGPT), Notion | Proposto | Pendente | |
| | Criação de Criativos e Copy | Geração de variações de copy, ideias de criativos, roteiros de vídeo. | LLM (Manus.im/ChatGPT), IA Generativa (Midjourney, DALL-E) | Proposto | Pendente | |
| | Otimização de Campanhas | Análise de performance, identificação de anomalias, sugestões de otimização. | LLM (Manus.im/ChatGPT), n8n | Proposto | Pendente | |
| | Criação de Sites/Landing Pages | Geração de código HTML/CSS/JS, conteúdo para seções. | Manus.im, LLM | Proposto | Pendente | |

| | Acompanhamento de Tarefas | Atualização automática de status, lembretes, criação de subtarefas. | Notion, n8n | Proposto | Pendente | |
| **Finalização/Renovação** | Análise de Performance e Resultados | Compilação de dados, geração de resumo executivo e insights. | n8n, LLM (Manus.im/ChatGPT) | Proposto | Pendente | |
| | Relatório de Resultados Abrangente | Geração automática de relatórios com dados e insights. | n8n, LLM, Looker Studio/Power BI | Proposto | Pendente | |
| | Proposta de Renovação | Geração de rascunhos de propostas personalizadas. | LLM (Manus.im/ChatGPT), Notion | Proposto | Pendente | |
| | Entrega de Acessos e Senhas | Criação de um documento consolidado de acessos para o cliente. | n8n, Notion | Proposto | Pendente | |
