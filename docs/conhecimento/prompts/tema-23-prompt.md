# Prompt de aplicação — Tema 23: Planejamento Estratégico de Longo Prazo com IA: sair do “apaga-incêndio” e usar IA como co-estrategista de 6–24 meses.

Use o prompt abaixo com IA para construir esse plano de 12 meses:

“Contexto: Sou gestor de tráfego, automações, criação de sites e IA. Quero que você atue como CO-ESTRATEGISTA DE LONGO PRAZO, não só como gerador de tarefas. Quero desenhar um plano de 12 meses que conecte minhas metas de negócio, meu portfólio de clientes e minhas capacidades de IA.

Visão para 12 meses (como quero estar atuando, tipo de clientes, ticket, proporção estratégia/operação, uso de IA):  
[descreva em 5–10 linhas].

Dados de base (opcional, mas ajuda):

  - nível atual de receita, nº de clientes, ticket médio, carga de trabalho;
  - serviços principais que ofereço hoje;
  - limitações de tempo/energia que preciso respeitar.

Quero que você:

1.  Transforme essa visão em 3–5 OBJETIVOS claros para 12 meses, e 3–5 KEY RESULTS mensuráveis para cada um (estilo OKR), usando faixas plausíveis.
2.  Liste as PRINCIPAIS INICIATIVAS necessárias para bater esses KRs (ofertas, melhorias internas, tipo de cliente a buscar, capacidades de IA a desenvolver).
3.  Monte um ROADMAP TRIMESTRAL (T1, T2, T3, T4) dizendo:
      
      - quais iniciativas entram em cada trimestre;
      - o que explicitamente fica de fora (para não sobrecarregar).  
        Leve em conta minha capacidade real e peça ajustes se você perceber overload.
4.  Proponha um SISTEMA DE ACOMPANHAMENTO com IA:
      
      - quais métricas acompanhar mensalmente;
      - que rituais (mensais/trimestrais) fazer comigo para revisar progresso e, se necessário, ajustar o plano;
      - que perguntas você deve me fazer nesses check-ins para atuar como ‘challenger’ da estratégia (apontando incoerências, riscos, oportunidades).

Responda específico para minha realidade (tráfego de performance, funis complexos, sites, IA) e evite generalidades de ‘tenha metas SMART’.”
