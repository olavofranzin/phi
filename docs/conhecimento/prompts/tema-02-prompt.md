# Prompt de aplicação — Tema 02: Reenquadramento Cognitivo com IA: usar “framing” para decidir melhor campanhas, funis e propostas.

Use o prompt abaixo em um assistente de IA, adaptando ao seu caso real:

“Contexto: Sou gestor de tráfego, automações e criação de sites. Quero usar você como parceiro de reenquadramento cognitivo (framing), não só como executor de tarefas.  
 Projeto: [descreva o cliente, oferta, ticket, canais de tráfego, principais automações e site atual].  
 Situação atual: [explique rapidamente os resultados atuais (KPI principais), o que fizemos até agora e o que me incomoda].  
 Meu frame atual do problema é: ‘[escreva em uma frase a forma como você enxerga hoje o problema central]’.

Quero que você:

1.  Gere pelo menos 7 frames alternativos para interpretar esse mesmo cenário, considerando ângulos de produto/mercado, mensagem, jornada, métricas, sazonalidade, comportamento da audiência e estratégia do cliente.
2.  Para cada frame, descreva quais ações principais ele sugeriria nos próximos 7–30 dias e quais métricas deveríamos observar para validar ou refutar esse frame.
3.  Me ajude a selecionar os 2 frames mais promissores, justificando a escolha, e transforme cada um em um mini-plano de ação enxuto (passos concretos) e em um parágrafo de explicação em linguagem leiga para meu cliente.  
     Seja específico para este projeto, evite conselhos genéricos de marketing e deixe explícitos os trade-offs de cada frame.”
